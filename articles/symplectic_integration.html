Symplectic integration is a numerical method for solving ordinary differential equations (ODEs) that is widely used in physics and astronomy. It is particularly well-suited for problems where the Hamiltonian is known and conserved, such as in celestial mechanics.

Recently I wrote a small planetary system simulator in Java Script that uses a symplectic integrator and it worked quite well: Planetary System Simulator.
